# 🎵 드럼 악보 뷰어 - 시작하기

환영합니다! 이 앱은 **유튜브 영상과 PDF 악보를 연동**하여 자동으로 스크롤되는 드럼 악보 뷰어입니다.

## ⚡ 3분 안에 시작하기

### 1️⃣ 웹 서버 실행 (필수)

이 폴더에서 터미널을 열고 아래 명령어를 실행하세요:

```bash
# Python이 있다면 (가장 쉬움)
python -m http.server 8000

# 또는 Node.js가 있다면
npx http-server -p 8000

# 또는 PHP가 있다면
php -S localhost:8000
```

### 2️⃣ 브라우저 열기

```
http://localhost:8000
```

### 3️⃣ 첫 곡 추가하기

1. **"+ 곡 추가"** 클릭
2. 곡 정보 입력
3. YouTube URL과 PDF 업로드
4. 타이밍 포인트 설정
5. **저장!**

## 🔥 주요 기능

### ✅ 곡 관리
- 제목, 아티스트, YouTube URL, PDF 저장
- 오프라인 저장 (IndexedDB)
- Export/Import로 백업

### ✅ 타이밍 설정
- YouTube 재생 시간 확인
- 원하는 시점에 포인트 추가
- 스크롤 위치 조정

### ✅ 악보 뷰어
- PDF 자동 렌더링
- YouTube와 동기화
- 타이밍에 맞춰 자동 스크롤
- 재생/일시정지 제어

### ✅ PWA 지원
- 홈 화면에 추가
- 오프라인 사용 가능
- 앱처럼 동작

## 💾 데이터 구조

```javascript

  title: "예뻤어",
```

## 🎯 사용 예시

### 곡 추가
```
1. 곡명: 예뻤어
2. 아티스트: DAY6
3. YouTube: https://www.youtube.com/watch?v=Mhyi9p7T7OM
4. PDF: 예뻤어_DAY6.pdf
5. 타이밍:
   - 0:15 → 500px
   - 0:45 → 1500px
   - 1:20 → 2500px
```

### 악보 보기
```
1. 메인 화면에서 곡 클릭
2. ▶ 재생 버튼 클릭
3. 자동 스크롤 확인!
```

### 백업
```
1. 우측 하단 ⋮ 클릭
2. Export 선택
3. JSON 파일 저장
```

## 🛠️ 문제 해결

### YouTube가 안 보여요
- 인터넷 연결 확인
- 올바른 URL 형식인지 확인
- `https://`가 포함되어 있는지 확인

### PDF가 안 보여요
- PDF 파일 크기 확인 (10MB 이하 권장)
- 다른 PDF 뷰어에서 정상 작동하는지 확인

### 자동 스크롤이 안 돼요
- 타이밍 포인트가 설정되어 있는지 확인
- 스크롤 값이 PDF 높이 내에 있는지 확인

## 💡 팁

### 정확한 타이밍 설정
- 곡을 여러 번 들으며 섹션 구분
- 4~8개 정도의 타이밍 포인트 권장
- 인트로, 벌스, 코러스, 브릿지 등

### PDF 최적화
- 고해상도 스캔 사용
- 파일 크기는 5~10MB 정도가 적당
- 가로 모드보다 세로 모드 권장

### 백업 습관
- 정기적으로 Export
- 클라우드 저장소에 JSON 파일 보관
- 다른 기기에서도 Import로 복구 가능
