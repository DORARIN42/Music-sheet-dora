## 🚀 빠른 시작 (3단계)

### 1️⃣ 웹 서버 실행

이 폴더에서 아래 명령어 중 하나를 실행하세요:

**Python이 설치되어 있다면:**
```bash
python -m http.server 8000
```

**Node.js가 설치되어 있다면:**
```bash
npx http-server -p 8000
```

**PHP가 설치되어 있다면:**
```bash
php -S localhost:8000
```

### 2️⃣ 브라우저에서 접속

```
http://localhost:8000
```

### 3️⃣ 첫 곡 추가하기

1. **"+ 곡 추가"** 버튼 클릭
2. 곡 정보 입력:
   - 곡명: `예뻤어`
   - 아티스트: `DAY6`
   - YouTube URL: `https://www.youtube.com/watch?v=Mhyi9p7T7OM`
   - PDF 파일: 업로드한 `예뻤어_DAY6.pdf` 선택

3. YouTube 영상을 재생하면서 타이밍 포인트 추가:
   - 첫 마디 시작: `0:15` → 스크롤: `500px`
   - 두 번째 섹션: `0:45` → 스크롤: `1500px`
   - 세 번째 섹션: `1:20` → 스크롤: `2500px`

4. **저장** 버튼 클릭

5. 메인 화면으로 돌아가서 곡을 클릭 → 자동 스크롤 테스트!

## 📱 PWA로 설치하기 (선택사항)

### Chrome/Edge (데스크톱)
1. 주소창 오른쪽의 **설치** 아이콘 클릭
2. "설치" 버튼 클릭
3. 앱이 별도 창으로 실행됩니다

### Chrome (모바일)
1. 우측 상단 메뉴(⋮) → **홈 화면에 추가**
2. 이름 확인 후 "추가" 클릭
3. 홈 화면에서 앱 아이콘으로 실행

### Safari (iOS)
1. 하단 공유 버튼 클릭
2. **홈 화면에 추가**
3. 이름 확인 후 "추가" 클릭

## ✅ 동작 확인 방법

### IndexedDB 확인
1. 개발자 도구 열기 (F12)
2. **Application** 탭 → **Storage** → **IndexedDB**
3. `DrumSheetDB` → `songs` 확인

### 데이터 백업
1. 메인 화면 우측 하단 **⋮** 버튼
2. **Export** 클릭
3. JSON 파일이 다운로드됨

### 데이터 복구
1. 메인 화면 우측 하단 **⋮** 버튼
2. **Import** 클릭
3. 백업한 JSON 파일 선택

## 🔧 문제 해결

### "서버를 시작할 수 없습니다"
- 포트가 이미 사용 중일 수 있습니다
- 다른 포트 번호를 시도하세요: `8080`, `3000`, `5000` 등

### "YouTube 플레이어가 로드되지 않습니다"
- 인터넷 연결 확인
- 올바른 YouTube URL 형식인지 확인:
  - ✅ `https://www.youtube.com/watch?v=VIDEO_ID`
  - ✅ `https://youtu.be/VIDEO_ID`
  - ❌ `www.youtube.com/...` (https:// 없음)

### "PDF가 표시되지 않습니다"
- PDF 파일이 손상되지 않았는지 확인
- PDF 파일 크기가 너무 크지 않은지 확인 (10MB 이하 권장)
- 다른 PDF 뷰어에서 정상적으로 열리는지 확인

### "자동 스크롤이 작동하지 않습니다"
- 타이밍 포인트가 올바르게 설정되었는지 확인
- 스크롤 위치가 PDF 높이보다 크지 않은지 확인
- 브라우저 콘솔(F12)에서 에러 메시지 확인

## 💡 팁

### 정확한 스크롤 위치 찾기
1. 곡 추가 페이지에서 PDF 업로드
2. "전체 PDF" 정보 확인 (예: 3페이지, 3,660px)
3. 페이지별 범위 확인 (예: 페이지 1: 0~1,200px)
4. 원하는 위치의 대략적인 px 값 계산

### 여러 타이밍 포인트 설정
- 곡의 구조에 맞춰 섹션별로 설정
- 인트로, 벌스, 코러스, 브릿지, 아웃트로 등
- 4~8개 정도가 적당함

### 백업 습관
- 곡을 여러 개 추가한 후 정기적으로 Export
- JSON 파일을 안전한 곳에 보관
- 다른 기기로 이동 시에도 Import로 복구 가능
